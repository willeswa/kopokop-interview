## Clone the project
- git clone https://github.com/willeswa/kopokop-interview.git
- cd kopokop-interview

## Set up the virtual environment
- python3.8 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt

## Run the script
- change variables in the for loop in the main.py file
- python main.py


## Assumptions
- This setup assumes you have a directory called "test_data" in the root directory of this project (i.e in the same level as main.py)
- The root directory should have atleast 1 .csv file
